# 03 教材大纲来源规则

## 芽芽助教项目背景

芽芽助教是一个面向 K12 全学段、多学科场景的自我进化 AI 教学适配引擎。它以真实教材知识图谱为骨架、以学习行为为真值、以错因诊断为核心、以策略调度为动作系统、以成长记忆为长期资产。

一句话：孩子用得越久，芽芽越懂他；芽芽教得越久，越知道怎么教他。

**当前建设范围**：人教版小学数学 1-6 年级全量覆盖。

**数据底座**：
- 394 个知识点（G1-G4: 205 个，G5-G6: 189 个），全量录入
- 86 个错因（L1 通用 10 个 + L2 数学特有 76 个），全量定义
- 1390 条知识点-错因绑定，394 个 KP 全覆盖
- 两层知识图谱（学习路径层 + 诊断依赖层），G1-G6 全部建成
- 题型规格矩阵（260KB）
- 数据质量门禁 data_quality_gate.py（9 铁规全覆盖）

**引擎架构**：诊断引擎（四步：初筛→匹配→验证→溯源）→ 策略引擎（六态调度）→ 家长摘要（中性安全反馈）→ 质量仪表盘（24 指标）。

**文档集**：14 份规划文档（01-14），覆盖产品愿景、开发路线、教材来源、分级规则、覆盖门禁、知识图谱、错因分类、策略包、事件协议、数据模型、诊断引擎、策略引擎、家长摘要、质量仪表盘。

## 本文档背景

这是芽芽助教产品文档集的第 03 号文件，定义知识点入库的数据来源规则和优先级（教材正文 > 教参 > 课标 > 教辅 > AI 整理）。它回答「这个知识点凭什么算数」——是后续所有知识点数据的权威性基础。本文档的直接产物是 primary_math_g1_g4_knowledge.json 和 primary_math_g5_g6_knowledge.json。

## 目标

在任何知识点、错因、策略开发前，必须先建立权威、完整、可追溯的人教版小学数学 1–6 年级知识底座。

## 必须采集的教材范围

### 五年级上册
- 小数乘法
- 位置
- 小数除法
- 可能性
- 简易方程
- 多边形的面积
- 数学广角
- 总复习

### 五年级下册
- 观察物体
- 因数与倍数
- 长方体和正方体
- 分数的意义和性质
- 图形的运动
- 分数的加法和减法
- 折线统计图
- 数学广角
- 总复习

### 六年级上册
- 分数乘法
- 位置与方向
- 分数除法
- 比
- 圆
- 百分数
- 扇形统计图
- 数学广角
- 总复习

### 六年级下册
- 负数
- 百分数（二）
- 圆柱与圆锥
- 比例

> 具体目录必须以实际人教版教材为准。以上只是预期范围，不是最终权威清单。

## 来源优先级

1. 人教版教材目录与正文
2. 人教版教师教学用书 / 教参
3. 义务教育数学课程标准
4. 同步教辅与真题作为辅助参考
5. AI 整理结果只作为草稿，不作为唯一来源

## 知识点入库要求

每个知识点必须记录来源：

```json
{
  "knowledgeCode": "M5S1-0",
  "name": "小数乘整数的意义",
  "grade": "5",
  "book": "上册",
  "unit": "第一单元 小数乘法",
  "lesson": "小数乘整数",
  "sourceType": "local_db_seed_plus_directory",
  "sourceRef": "人教版五年级上册第X页",
  "verifiedBy": "human | ai_assisted | pending",
  "confidence": 1.0
}
```

## 待审核池规则

如果知识点由 AI 推断得出，但没有明确教材 / 教参 / 课标来源：

- 不得直接入正式库。
- 必须进入待审核池。
- 待补来源后才能参与 CoverageGate。
