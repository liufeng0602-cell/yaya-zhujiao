# 人教版小学数学 1-4 年级教材大纲

> 来源: 2022版义务教育数学课程标准

> 状态: 初稿


## 一年级上册

### 数与代数

**第1单元 准备课**

- [C][课标了解] 了解数一数、实物对应数数方法
- [B][课标理解] 理解同样多、多、少的含义
- [B][课标掌握] 掌握一一对应比较多少的方法
**第2单元 1~5 的认识和加减法**

- [B][课标理解] 理解 1-5 各数的含义、读写与顺序
- [B][课标掌握] 掌握 1-5 数的分与合
- [A][课标理解] 理解加减法的实际含义
- [A][课标掌握] 掌握 1-5 以内加减法口算
**第3单元 0 的认识和加减法**

- [B][课标了解] 了解 0 的两种实际含义
- [A][课标掌握] 掌握 0 的读写与有关 0 的加减法
**第4单元 20 以内的进位加法**

- [B][课标理解] 理解凑十法算理
- [A][课标掌握] 掌握 9、8、7、6 加几进位口算
- [A][课标掌握] 掌握 5、4、3、2 加几口算技巧
- [A][课标运用] 运用进位加法解决生活简单问题
### 图形与几何

**第5单元 认识立体图形**

- [C][课标了解] 了解长方体、正方体、圆柱、球的外形特征
- [B][课标掌握] 能直观辨认、分类常见立体图形
### 常见的量

**第6单元 认识钟表**

- [C][课标了解] 了解钟面时针、分针
- [B][课标掌握] 掌握认读整时的方法
- [B][课标掌握] 会用两种形式记录整时
### 综合与实践

**第7单元 数学广角**

- [C][课标了解] 发现简单图形、数字排列规律
- [C][课标掌握] 能按规律补全序列
